# C Hello World

* **Compilador**: GCC Rev13
* **Versión**: 15.2.0
* **Versión de C**: C23

## Resolución
Se utilizó VSCode para escribir el código, y la terminal dentro del mismo en donde se escribieron los siguientes comandos:
```bash
gcc hello.c
./a
```
El primer comando compila el código en `hello.c` y genera un ejecutable nombrado `a.exe`, el cual es ejecutado por el segundo comando.
